## Install and use

- 技术文档
  [http://vue3.jeecg.com](http://vue3.jeecg.com)
  
  
- Get the project code

```bash
git clone https://github.com/jeecg/vue3-jeecg-admin.git
```

- Installation dependencies

```bash
cd vue3-jeecg-admin

yarn install

```

- run

```bash
yarn serve
```

- build

```bash
yarn build
```
