#### Install and use

- 技术文档

  [http://vue3.jeecg.com](http://vue3.jeecg.com)
  
- 在线演示（beta版本）

  [http://boot3.jeecg.com](http://boot3.jeecg.com)

  
- Get the project code

```bash
git clone https://github.com/jeecgboot/jeecg-boot-vue3.git
```

- Installation dependencies

```bash
cd jeecg-boot-vue3

yarn install

```

- run

```bash
yarn serve
```

- build

```bash
yarn build
```
